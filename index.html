# (https://github.com/jcangas/PureMVC4D">PureMVC4D)by [jcangas](https://github.com/jcangas)
PureMVC is a lightweight framework for creating applications based upon the classic Model-View-Controller design meta-pattern. This is the specific implementation for the Delphi language, based in a previous C# port by Andy Adamczak
